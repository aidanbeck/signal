A game made for the 59th Ludum Dare Game Jam.
Theme: "Signal".

Equipped with sonar technology, you are tasked with searching a dark ocean for a nefarious submarine. Receive signals on your computer for clues of it’s location, and deduce where it lies by drawing on a map.
You have three attempts to fire upon it’s location before it gets to you first.
